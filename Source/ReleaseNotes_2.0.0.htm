Change Log

  • Please Note: This version includes breaking changes to previous versions of Engage: Rotator
  • The display is now controlled through HTML-based templates
  • After upgrading from a previous version, the display of the rotator content (slides) will be different
  • Added many more options for the rotation
  • Many new animation types
  • The ability to display slides in a random order
  • Fixed a bug where slides were stored by TabModuleID, meaning that copying an existing Rotator module onto a different page would not copy the slides
  • More details can be found on the Engage: Rotator Change Log

Known Issues

  • Engage: Events 1.1 is incompatible with Engage: Rotator 2.0.  Please upgrade to Engage: Events 1.2 after installing Engage: Rotator 2.0.
  • Datepicker on settings page displays inconsistently in some skins
